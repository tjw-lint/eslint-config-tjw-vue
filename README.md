# eslint-config-tjw-vue

The Jared Wilcurt's Vue.js ESLint rules


## Using this

1. `npm install --save-dev eslint-config-tjw-vue`
1. In your `.eslitrc.js` add `tjw-vue` to your `extends` like so:
    ```js
    module.exports = {
      'extends': [
        'tjw-vue'
      ]
    };
    ```
